

# Scheduled Display System #
## Features

- User authentication (login system)
- Session-based access control
- Add and manage scheduled content
- Display data dynamically from database
- Modular PHP structure
- Clean separation of frontend and backend logic

## Database

- The project uses MySQL for data storage
- Database schema is provided in `database.sql`
- Includes tables for users and scheduled content management

## Key Learning Outcomes

- PHP backend development
- MySQL database integration
- Session management in web applications
- CRUD operations implementation
- Structuring a full-stack web project
